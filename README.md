# React Query + JSON Server CRUD app

Test project using React Query and JSON Server to create a sample CRUD app which allows to add, remove and modify a list of products.

## Technologies used

Built with:

- Vite + React
- JavaScript
- React Query
- JSON Server

## Setup and usage

`npm install`

`npm run back`

`npm run dev`
